# BEASTMODE PROMPTS (BEASTMODE TIER)

These 30 advanced prompts are designed to leverage the SML-RECALL Beastmode scripts, maximizing Claude's analytical and organizational capabilities.

## Context Engine & Routing
1. "Run `beastmode/context-engine.py --task 'refactor database layer'` and use the output to guide my next 3 steps."
2. "I need to switch context to Project X. Run `beastmode/thread-router.py --task 'Project X'` and give me the optimal starting point."
3. "Extract the relevant context blob for 'authentication rewrite' using `context-engine.py` and identify any missing architectural decisions."
4. "Using the output from `thread-router.py --task 'API rate limiting'`, write a technical specification for the new feature."
5. "Feed the output of `context-engine.py --project 'frontend'` into a fresh session and summarize the current state of the UI."

## Impact Analysis
6. "Run `beastmode/impact-analysis.py --change 'migrating from SQLite to PostgreSQL'` and detail the direct and indirect impacts."
7. "Generate a pre-change checklist using `impact-analysis.py` for 'updating React to version 19'."
8. "Based on the risk level output from `impact-analysis.py --change 'rewrite auth middleware'`, should we proceed today or wait?"
9. "Identify all safe items from `impact-analysis.py --change 'delete legacy user table'` and draft the SQL migration for them."
10. "Run an impact analysis on 'changing default theme to dark mode' and tell me which files in `project-context.md` will need updates."

## Health Scoring & Maintenance
11. "Run `beastmode/health-score.py` and explain why my 'completeness' score dropped since last week."
12. "Using the output of `health-score.py --json`, generate a step-by-step plan to improve my 'decision coverage' to an A grade."
13. "Poll `health-score.py --watch` while we work and alert me if 'freshness' drops below a B."
14. "My 'actionability' score is a C. Run `health-score.py` and tell me which tasks in `MASTER.md` are dragging it down."
15. "Compare today's `health-score.py` output with last week's and highlight the areas where my documentation has degraded."

## Auto-Updating & Stale Content
16. "Run `beastmode/auto-update.py --days 14` and walk me through answering the generated questions for stale projects."
17. "Review the `update-queue.md` generated by `auto-update.py` and draft suggested answers based on our recent chat history."
18. "Execute `auto-update.py` to find any projects in `MASTER.md` that haven't been touched in 7 days, and propose a revival strategy."
19. "Use `auto-update.py` to identify neglected areas of my codebase and generate a 2-hour focused sprint to catch up."
20. "Run `auto-update.py --days 30`. For any project older than 30 days, draft a formal 'sunset' or 'pause' entry for the decision log."

## Thread Organization
21. "Run `beastmode/thread-organizer.py --clean` and summarize the ACTIVE threads that remain in `thread-tracker.md`."
22. "Execute `thread-organizer.py --archive` and write a brief retrospective on the threads that were just archived."
23. "Use `thread-organizer.py` to identify BLOCKED threads and suggest 3 unblocking actions for the most critical one."
24. "Run `thread-organizer.py` to classify all threads. Are there more STALE threads than ACTIVE ones? If so, why?"
25. "After running `thread-organizer.py --clean`, rewrite my `MASTER.md` immediate focus to align with the remaining active threads."

## Cross-Script Synergies
26. "Run `health-score.py` to find the weakest area, then run `context-engine.py` on that specific project to figure out what's missing."
27. "Execute `impact-analysis.py` for 'adding Stripe integration', then run `thread-router.py` to setup my workspace for the change."
28. "Use `auto-update.py` to gather stale context, then run `health-score.py` to see if answering those questions improves the grade."
29. "Run `thread-organizer.py --archive`, then use `context-engine.py --task 'archived items'` to generate a monthly progress report."
30. "Generate a comprehensive daily briefing by combining the outputs of `health-score.py`, `thread-organizer.py --clean`, and `auto-update.py`."
